Detailed Description

SatOrbiterNetDeviceDvb to be utilized in geostationary satellite.

SatOrbiterNetDeviceDvb holds a set of phy layers towards user and feeder links; one pair of phy layers for each spot-beam. The SatNetDevice implements a simple switching between all user and feeder links modeling transparent payload.
